# Ness — `ness.city`

The civic layer for **Network School**.

The core team can't see every problem. Ness is where citizens surface issues, name root causes, ship fixes, and earn social credit for the documentation they leave behind.

## Three loops

1. **Surface** — anyone files a problem with a real diagnosis (+5 credit).
2. **Solve** — a citizen ships a fix and documents it (+25 credit, +8 per upvote).
3. **Remember** — every documented solution becomes part of the city's memory.

## Stack

- Next.js 15 (App Router) · React 19 · TypeScript
- TailwindCSS 3
- Deployed on Vercel

## Local dev

```bash
npm install
npm run dev
```

Open http://localhost:3000.
